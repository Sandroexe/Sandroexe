Dieses Projekt dient als meine digitale Visitenkarte und zentraler Hub für alle meine Profile.

## 🚀 Über das Projekt
Die Webseite ist eine minimalistische "Link-in-Bio" Lösung, die direkt auf GitHub Pages gehostet wird. Sie bietet Besuchern einen schnellen Überblick über meine Kontaktmöglichkeiten und sozialen Netzwerke.

**Highlights:**
* 📱 **Full Responsive:** Optimiert für Smartphones, Tablets und Desktop-PCs.
* 🌑 **Modern Dark Mode:** Ein cleanes Design, das die Augen schont.
* ⚡ **Performance:** Extrem schnelle Ladezeiten durch Verzicht auf schwere Frameworks.
* ⚖️ **Rechtssicher:** Vollständiges Impressum und Datenschutzerklärung nach österreichischem Recht.

## 🛠 Tech Stack
* **Frontend:** HTML5 & CSS3 (Flexbox & SVG Icons)
* **Hosting:** GitHub Pages
* **Domain:** exemail.at (via INWX)
